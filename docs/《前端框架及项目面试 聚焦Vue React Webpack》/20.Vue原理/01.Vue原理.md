## Vue原理(大厂必考)

* 面试为何会考察原理 ?
  * 之前然知其所以然一各行业通 用的道理
  * 了 解原理,才能应用的更好(竞争激烈,择优录取)
  * 大厂造轮子(有钱有资源,业务定制,技术KPI)

* 面试中如何考察?以何种方式?
  * 考察重点，而不是考察细节。掌握好2/8原则
  * 和使用相关联的原理,例如vdom、模板渲染
  * 整体流程是否全面?热门技术是否有深度?

* Vue 原理包括哪些?
  * 组件化
  * 响应式
  * vdom和diff
  * 模板编译
  * 渲染过程
  * 前端路由
